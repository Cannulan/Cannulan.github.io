## Java 接入有多简单——直接抄示例

Mistral 官方 Java SDK 需要额外引入依赖、配置环境变量，对很多已经跑着 OpenAI SDK 的项目来说，改起来挺麻烦。

千聚的接入方式让这件事变得极其简单：**改用千聚的 base_url，API key 一换，代码就能跑。**

java
import okhttp3.*;
import com.google.gson.JsonObject;
import com.google.gson.JsonParser;

public class MistralLargeExample {
    public static void main(String[] args) throws Exception {
        // 原来是用 Mistral 官方 base_url
        // String baseUrl = "https://api.mistral.ai/v1";
        
        // 换成千聚的 API 地址，一行改动
        String baseUrl = "https://www.qianjuai.com/v1";
        String apiKey = "你的千聚API Key";  // 从千聚后台获取

        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
                .build();

        JsonObject messages = new JsonObject();
        messages.addProperty("role", "user");
        messages.addProperty("content", "用Java写一个快速排序算法");

        JsonObject body = new JsonObject();
        body.addProperty("model", "mistral-large-latest");
        body.add("messages", new com.google.gson.JsonArray());
        body.getAsJsonArray("messages").add(messages);
        body.addProperty("max_tokens", 4096);
        body.addProperty("temperature", 0.7);

        Request request = new Request.Builder()
                .url(baseUrl + "/chat/completions")
                .addHeader("Authorization", "Bearer " + apiKey)
                .addHeader("Content-Type", "application/json")
                .post(RequestBody.create(
                        MediaType.parse("application/json"),
                        body.toString()
                ))
                .build();

        Response response = client.newCall(request).execute();
        String responseBody = response.body().string();
        JsonObject jsonResponse = JsonParser.parseString(responseBody).getAsJsonObject();
        String content = jsonResponse
                .getAsJsonArray("choices")
                .get(0).getAsJsonObject()
                .getAsJsonObject("message")
                .get("content")
                .getAsString();

        System.out.println("Mistral Large 返回：\n" + content);
    }
}


**核心改动就两点**：
1. `baseUrl` 改为 `https://www.qianjuai.com/v1`
2. `apiKey` 换成千聚后台申请的 key

其他逻辑一概不变。你的 Spring Boot 项目、LangChain4j、Spring AI 等框架，只需要在配置文件中把 API 地址和 key 替换一下，就能直接使用 Mistral Large。
