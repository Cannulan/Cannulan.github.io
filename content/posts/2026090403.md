### 第二步：写核心调用代码（兼容Mistral Large）

创建一个Java类，比如叫 `MistralLargeClient`，核心代码很少：

java
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.service.OpenAiService;
import java.time.Duration;
import java.util.List;

public class MistralLargeClient {
    public static void main(String[] args) {
        // 1. 设置API Key和Base URL
        String apiKey = "这里替换成你在千聚申请的API Key";
        String baseUrl = "https://www.qianjuai.com/v1";

        // 2. 创建OpenAiService实例
        OpenAiService service = new OpenAiService(baseUrl, apiKey, Duration.ofSeconds(60));

        // 3. 构建请求，指定模型为Mistral Large
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("mistralai/mistral-large-2407")
                .messages(List.of(
                        new ChatMessage("user", "请用Java写一个斐波那契数列生成器")
                ))
                .maxTokens(1000)
                .temperature(0.7)
                .build();

        // 4. 发送请求并打印结果
        service.createChatCompletion(request)
                .getChoices()
                .forEach(choice -> System.out.println(choice.getMessage().getContent()));
    }
}


代码注释里已经标清楚重点：第2行改Base URL，第3行定Key，第10行指定模型名。别的都是OpenAI的那一套。
