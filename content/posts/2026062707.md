### 第三步：写一个 Gemini Pro 调用的 Java 方法

下面这个例子，使用 HttpsURLConnection 做纯原生调用，你看完可以直接复制。

java
import java.io.*;
import java.net.HttpURLConnection;
import java.net.URL;
import java.nio.charset.StandardCharsets;

public class GeminiProExample {

    private static final String API_KEY = "sk-你从千聚申请的key";  
    private static final String API_URL = "https://www.qianjuai.com/v1/chat/completions";
    
    public static void main(String[] args) {
        try {
            String response = callGeminiPro("用一句中文介绍Google Gemini Pro的能力");
            System.out.println("响应：\n" + response);
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
    
    public static String callGeminiPro(String userMessage) throws IOException {
        URL url = new URL(API_URL);
        HttpURLConnection con = (HttpURLConnection) url.openConnection();
        con.setRequestMethod("POST");
        con.setRequestProperty("Authorization", "Bearer " + API_KEY);
        con.setRequestProperty("Content-Type", "application/json");
        con.setDoOutput(true);
        
        // 构建请求体（使用 OpenAI 聊天格式，但背后调用 Gemini Pro 1.5）
        String body = "{\n"
                + "  \"model\": \"gemini/gemini-1.5-pro\",\n"
                + "  \"messages\": [{\n"
                + "    \"role\": \"user\",\n"
                + "    \"content\": \"" + userMessage + "\"\n"
                + "  }]\n"
                + "}";
        
        try(OutputStream os = con.getOutputStream()) {
            byte[] input = body.getBytes(StandardCharsets.UTF_8);
            os.write(input, 0, input.length);
        }
        
        int status = con.getResponseCode();
        if (status == 200) {
            BufferedReader in = new BufferedReader(new InputStreamReader(con.getInputStream()));
            String inputLine;
            StringBuffer content = new StringBuffer();
            while ((inputLine = in.readLine()) != null) {
                content.append(inputLine).append("\n");
            }
            in.close();
            return content.toString();
        } else {
            // 错误处理
            BufferedReader in = new BufferedReader(new InputStreamReader(con.getErrorStream()));
            String inputLine;
            StringBuffer content = new StringBuffer();
            while ((inputLine = in.readLine()) != null) {
                content.append(inputLine);
            }
            in.close();
            throw new IOException("Api response: " + status + " error: " + content.toString());
        }
    }
}


**这个核心逻辑是**：将你原先面向 OpenAI 的“聊天问答”格式（`model`/`messages`）保持不变，只是 model 名称写为 `"gemini/gemini-1.5-pro"`，接着用自己的 `API_KEY`，千聚后台会自动路由到 Google 的 Gemini Pro 1.5 模型。

> 如果你用的是 Spring Boot 的 RestTemplate：完全同理，只换 URL 和加认证 Header。
