## Java示例接入：改一行代码，成本砍半

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import java.util.List;

public class DeepSeekJavaExample {

    public static void main(String[] args) {
        // 原先是 api.openai.com，现在换成千聚的地址
        String baseUrl = "https://www.qianjuai.com/v1";
        // 替换成你在千聚后台申请的API Key（从限时特价分组中获取）
        String apiKey = "sk-你的千聚APIKey";

        OpenAiService service = new OpenAiService(baseUrl, apiKey);

        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("deepseek-chat") // DeepSeek-V3的模型名
                .messages(List.of(
                        new ChatMessage("system", "你是一个Java编程专家。"),
                        new ChatMessage("user", "请用Java写一个快速排序算法的实现。")
                ))
                .maxTokens(2000)
                .temperature(0.2)
                .build();

        service.createChatCompletion(request)
                .getChoices()
                .forEach(choice -> {
                    System.out.println(choice.getMessage().getContent());
                });
    }
}

1. **baseUrl**：这是最关键的一行。`https://www.qianjuai.com/v1`完全兼容OpenAI的API端点，所以SDK直接支持。
2. **API Key**：去[千聚ai大模型聚合站](https://www.qianjuai.com/)注册后，在“限时特价”分组里生成一个新Key。注意，不同分组对应不同的Key，别混用。
3. **模型名**：`deepseek-chat`就是DeepSeek-V3的标准模型名。
