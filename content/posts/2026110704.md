## 流式输出、多模态、工具调用——全支持

千聚的 API 完整兼容 OpenAI 格式，不仅普通文本对话能用，流式输出、多模态输入、Function Calling 这些高级特性也全部支持。

**流式输出示例：**

javascript
async function streamCall(modelName, userMessage) {
  const stream = await client.chat.completions.create({
    model: modelName,
    messages: [
      { role: 'user', content: userMessage },
    ],
    stream: true,                  // 开启流式
  });

  for await (const chunk of stream) {
    process.stdout.write(chunk.choices[0]?.delta?.content || '');
  }
}

// 流式调用 GPT-4.1
streamCall('gpt-4.1', '用流式方式给我讲一个笑话');


**多模态（图片+文本）示例：**

javascript
async function visionCall(modelName, imageUrl) {
  const completion = await client.chat.completions.create({
    model: modelName,
    messages: [
      {
        role: 'user',
        content: [
          { type: 'text', text: '这张图片里是什么内容？' },
          {
            type: 'image_url',
            image_url: { url: imageUrl },
          },
        ],
      },
    ],
  });

  return completion.choices[0].message.content;
}
