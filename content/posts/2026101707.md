### 方式一：直接调用[千聚ai聚合站](https://www.qianjuai.com/) API（最推荐）

java
import okhttp3.*;
import org.json.JSONObject;

public class GPT5MiniExample {
    public static void main(String[] args) throws Exception {
        String apiKey = "你的[千聚ai聚合站](https://www.qianjuai.com/)API Key";
        String url = "https://www.qianjuai.com/v1/chat/completions";

        JSONObject requestBody = new JSONObject();
        requestBody.put("model", "gpt-5-mini");
        requestBody.put("messages", new JSONArray()
                .put(new JSONObject().put("role", "user").put("content", "Hello, GPT-5mini!")));

        OkHttpClient client = new OkHttpClient();
        Request request = new Request.Builder()
                .url(url)
                .header("Authorization", "Bearer " + apiKey)
                .post(RequestBody.create(MediaType.parse("application/json"), requestBody.toString()))
                .build();

        Response response = client.newCall(request).execute();
        System.out.println(response.body().string());
    }
}

**费用：** 0 元起步（新用户有免费额度），后续按 GPT-5mini 官方费率 * 1 倍计算
