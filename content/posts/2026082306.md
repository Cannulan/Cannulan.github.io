## 流式输出（Stream）示例

如果你的应用需要实时展示模型输出（比如聊天机器人），流式输出是更好的选择。以下是流式调用文心一言的示例：

javascript
// stream-example.js
const OpenAI = require('openai');

const client = new OpenAI({
  baseURL: 'https://www.qianjuai.com/v1',
  apiKey: 'sk-你的千聚API Key'
});

async function streamCall() {
  const stream = await client.chat.completions.create({
    model: 'wenxin',
    messages: [
      { role: 'user', content: '给我讲一个关于程序员的笑话，要短的。' }
    ],
    stream: true,  // 开启流式输出
    temperature: 0.9
  });

  console.log('🤖 文心一言：');
  for await (const chunk of stream) {
    const content = chunk.choices[0]?.delta?.content || '';
    process.stdout.write(content);  // 实时输出
  }
  console.log('\n✅ 流式输出完成');
}

streamCall().catch(console.error);


流式模式下，文心一言会逐字返回内容，体验非常流畅。而且这一切都**在国内直连环境中**完成，没有任何网络延迟的焦虑。
