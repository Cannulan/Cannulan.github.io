## 进阶操作：支持流式输出和连续对话

如果你想让模型像 ChatGPT 一样一个字一个字地“蹦”出来，而不是一次性输出完，可以开启流式输出。

python
from openai import OpenAI

client = OpenAI(
    base_url="https://www.qianjuai.com/v1",
    api_key="sk-xxxxx"
)

# 流式输出
stream = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=[{"role": "user", "content": "用10个字描述一下编程"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")


> 这串代码会让模型一个字一个字地在你的终端里打印出来，效果很炫酷。
