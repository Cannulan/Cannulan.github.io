### 场景4：文件上传和图片理解（视觉能力）

智谱清言的部分模型支持图片理解，这段代码是调用示例。

python
import os
import base64
from openai import OpenAI

client = OpenAI(
    api_key="你的[千聚api中转站](https://www.qianjuai.com/)API Key",
    base_url="https://www.qianjuai.com/v1"
)

# 假设你有一张图片，将其编码为base64
def encode_image(image_path):
    with open(image_path, "rb") as image_file:
        return base64.b64encode(image_file.read()).decode('utf-8')

image_path = "你的图片路径.jpg"
base64_image = encode_image(image_path)

response = client.chat.completions.create(
    model="glm-4v",  # 注意：带视觉能力的模型名称
    messages=[
        {
            "role": "user",
            "content": [
                {
                    "type": "text",
                    "text": "这张图片里有什么？帮我描述一下。"
                },
                {
                    "type": "image_url",
                    "image_url": {
                        "url": f"data:image/jpeg;base64,{base64_image}"
                    }
                }
            ]
        }
    ],
    max_tokens=300
)

print(response.choices[0].message.content)


**解读**：需要图片理解时，调用带v后缀的模型（如glm-4v），以多模态格式传入图片。[千聚api中转站](https://www.qianjuai.com/)完美支持这种OpenAI格式的多模态调用。
