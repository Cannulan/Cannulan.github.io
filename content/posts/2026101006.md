### 第四步：避坑指南——这些错误千万别犯

#### 坑一：Base URL 后面乱加路径

> 错误：`https://www.qianjuai.com/v1/chat/completions`
> 正确：`https://www.qianjuai.com/v1`

SDK 会自动补全路径，你填完整路径反而会报错。

#### 坑二：API Key 复制多了空格

复制 API Key 的时候，注意别把前后的空格带进去。一旦有空格，服务端会直接返回认证失败。

#### 坑三：模型名称写错了

Claude Opus 的完整模型 ID 是 `claude-3-opus-20240229`，不是 `claude-opus` 也不是 `opus`。大小写也要严格区分，写错了系统根本不认识。

#### 坑四：忘了勾选“流式输出”

如果你想要逐字输出的效果（类似打字机），记得在请求里加上 `stream: true` 参数。[千聚ai聚合站](https://www.qianjuai.com/)支持流式响应，不勾的话会一次性返回全部结果，体验差不少。
