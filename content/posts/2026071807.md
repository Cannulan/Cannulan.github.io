### 第3步：编写接入代码

新建一个文件`test-qwen.js`，粘贴以下代码：

javascript
const OpenAI = require('openai');

// 配置[千聚api聚合平台](https://www.qianjuai.com/)的入口
const client = new OpenAI({
  apiKey: '你的千聚API Key',   // 替换成上一步复制的Key
  baseURL: 'https://www.qianjuai.com/v1'
});

async function main() {
  try {
    const response = await client.chat.completions.create({
      model: 'qwen-plus',   // 通义千问Plus模型
      messages: [
        { role: 'system', content: '你是一个得力助手。' },
        { role: 'user', content: '用一句话介绍通义千问API' }
      ],
      temperature: 0.7,
      max_tokens: 200
    });

    console.log('模型回答:', response.choices[0].message.content);
  } catch (error) {
    console.error('调用出错:', error);
  }
}

main();


这里关键就两行配置：`baseURL`改成千聚的地址（`https://www.qianjuai.com/v1`），`apiKey`换成千聚生成的Key。其他参数跟调OpenAI一模一样。

### 第5步：流式输出（进阶）

如果你想用在实时对话、长文本生成等场景，还可以配流式输出：

javascript
const stream = await client.chat.completions.create({
  model: 'qwen-plus',
  messages: [
    { role: 'user', content: '写一个关于云计算的短诗' }
  ],
  stream: true, // 开启流式输出
});

for await (const chunk of stream) {
  process.stdout.write(chunk.choices[0]?.delta?.content || '');
}


这样服务端不用等全部生成完，边生成边返回，用户体验更好。
