## Node.js接入示例：改一行代码的事

好，前面说再多，不如上手跑一遍代码。接Qwen-VL其实特别简单，尤其千聚兼容了OpenAI的接口格式。

你原来用官方Qwen-VL API时，代码里大概是这样的：

    apiKey: '你的阿里云DashScope API Key',

现在换成千聚，就改两个地方：

    apiKey: '你在千聚申请的API Key',
    baseURL: 'https://www.qianjuai.com/v1' // 重点：指向千聚接口
});


然后调用Qwen-VL（以Qwen-VL-Max版本为例）的方式完全一样：

javascript
async function analyzeImage(imageUrl) {
    const response = await client.chat.completions.create({
        model: "qwen-vl-max", // 模型名称，千聚完全兼容
        messages: [
            {
                role: "user",
                content: [
                    { type: "text", text: "请帮我识别这张图片里的所有商品名称和价格，并整理成表格返回。" },
                    { type: "image_url", image_url: { url: imageUrl } }
                ]
            }
        ]
    });
    console.log(response.choices[0].message.content);
}
