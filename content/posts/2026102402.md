## 第四步：扩展一下，让它更像一个“对话”

光跑一次不够过瘾。我们来加个简单的对话循环，让你可以和AI持续聊天。

新建一个 `dialogue.js` 文件，粘贴以下代码：

javascript
import OpenAI from 'openai';
import readline from 'readline';

const openai = new OpenAI({
  apiKey: '你的千聚API Key',
  baseURL: 'https://www.qianjuai.com/v1'
});

const rl = readline.createInterface({
  input: process.stdin,
  output: process.stdout
});

const messages = [
  { role: 'system', content: '你是一个幽默风趣的问答助手，回答尽量简短。' }
];

async function askAI() {
  rl.question('你: ', async (input) => {
    if (input.toLowerCase() === 'exit') {
      rl.close();
      return;
    }

    messages.push({ role: 'user', content: input });

    const completion = await openai.chat.completions.create({
      model: 'gpt-5mini',
      messages: messages,
      max_tokens: 300
    });

    const reply = completion.choices[0].message.content;
    console.log('AI:', reply);
    messages.push({ role: 'assistant', content: reply });
    askAI();
  });
}

askAI();


运行 `node dialogue.js`，你会进入一个交互式对话界面。输入你想聊的内容，按回车，AI会回复你。输入 `exit` 退出。

这就是一个最简单版本的智能聊天机器人了。你可以把 `system` 角色的提示词改成“你是客服”、“你是翻译助手”或者“你是你的私人写作助理”，立刻就变成不同的应用场景。
